用PC的chrome浏览器调试手机的Safari页面

Posted on 2017-03-07 | In 工具 | Visitors

背景

在Mac上用safari调试手机页面，有个弊端，就是第一次的console.log和其他的记录是捕捉不到的，而且习惯用chrome调试的同学对safari无感，所以 ios-webkit-debug-proxy诞生了

安装

brew install ios-webkit-debug-proxy
brew uninstall –force libimobiledevice ios-webkit-debug-proxy
brew install –HEAD libimobiledevice ios-webkit-debug-proxy

连接手机

设置 > Safari > 高级 > Web 检查器 > 打开

启动代理

terminal执行

ios_webkit_debug_proxy -f chrome-devtools://devtools/bundled/inspector.html

pc的chrome浏览器启动
